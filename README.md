# ScreenshotTool

```
请从零生成一个完整可运行的 macOS 项目，目标是做一个“截图工具”，严格按下面规格输出完整项目文件内容。

目标功能
1. 这是一个 macOS  App，打开后会有一个图标显示在顶部菜单栏。
2. 点击菜单栏的图标，会显示一个列表，显示区域截图和全屏截图按钮，设置和退出按钮。
3. 区域截图和全屏截图按钮显示为一组，设置和退出显示为一组。
4. 鼠标移动到设置按钮上会弹出二级菜单，显示“生成截图到桌面”，是一个 switch 组件。
5. 点击截图按钮后，进入截图模式，这时鼠标按住作为截图的一个顶点，鼠标松开作为另一个顶点。松开后把两个顶点组成的矩形作为截图。
6. 自动把截图的图片数据复制到剪切板，方便后续粘贴
7. 如需保存截图到桌面，文件命名为shot-(YYYY-MM-DD-HH-mm-ss).png

技术要求
1. 使用 Swift Package Manager。
2. 平台必须是 macOS 13+。
3. 最终通过 Makefile 组装成 .app + .appex 结构，而不是 Xcode 工程。
4. 必须生成完整文件，不要只给片段。
5. 如果项目文件已存在，则列出目标功能中哪些完成了，哪些没完成，把没完成的功能做完。以尽量少的改动完成功能。
```
